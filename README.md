Description

A creative text generation application that combines random words from curated word banks to create poetic, meaningful-sounding sentences. Inspired by "answer books" and creative writing prompts.
Features

    ✅ Multiple word banks (starters, subjects, verbs, adjectives, endings)

    ✅ Random sentence generation

    ✅ Two generation modes (simple and poetic)

    ✅ Beautiful book-themed UI

    ✅ Keyboard support (press space for new answer)

    ✅ Fade animations

    ✅ Responsive design

Technologies Used

    HTML5

    CSS3 (3D transforms, animations)

    JavaScript (Arrays, randomization, template literals)

Learning Outcomes

    Array manipulation

    Random number generation

    Template literals

    Creative coding concepts

    Event handling (click and keyboard)
